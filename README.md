### What is ORIGIN Protocol?

A decentralized data ownership and compensation infrastructure built on Ethereum Layer 2 (Base network). Every creator — writer, artist, developer, musician — can register cryptographic proof of their work on-chain. When AI companies train on registered content, license fees flow automatically and directly to creators.

**Token:** $OGN  
**Chain:** Base (Ethereum L2)  
**Status:** Pre-launch — whitepaper phase
